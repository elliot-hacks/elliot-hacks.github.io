What is CEH?

The Certified Ethical Hacker (CEH) is a qualification offered by the EC-Council that certifies individuals in the specific network security discipline of ethical hacking. The certification is designed to provide a comprehensive understanding of:

  • Ethical hacking methodologies and frameworks
  • Vulnerability assessment and penetration testing
  • Network and web application security
  • Malware analysis and reverse engineering
  • Social engineering techniques
  • Mobile and IoT device security
  • Cloud computing security
